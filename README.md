# Termux Setup Tools

Quick setup scripts for configuring Termux environment.

## Usage

Run the setup script with one command:

```bash
bash install.sh
```

Or directly:

```bash
curl -sL https://raw.githubusercontent.com/Diko8662/dkbhaipro/main/install.sh | bash
```

## Features

- Auto-installs common packages
- Configures environment variables  
- Sets up useful aliases
- Optimizes terminal settings

## Requirements

- Termux app installed
- Internet connection
